# Contributing

If you have cool stuff you want added, feel free to open an issue and I'll try to add it. Otherwise, you can implement it yourself. I apologize for the messy code and appreciate your interest in the project!

### Bug reporting
Make an issue please or contact `slothh__` on Discord. The more detailed you can be, the better. Screenshots, logs, data/mappings, and steps on how to reproduce the issue would be greatly appreciated.

### No AI-generated code
Orpheus is entirely written by humans and it would be nice to keep it that way.
